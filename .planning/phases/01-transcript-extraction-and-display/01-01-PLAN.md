  <behavior>
    ## timestampToSeconds
    - "1:23" -> 83
    - "1:23:45" -> 5025
    - "0:00" -> 0
    - "10:05" -> 605

    ## formatTimestamp
    - 83 -> "1:23"
    - 5025 -> "1:23:45"
    - 0 -> "0:00"
    - 61 -> "1:01"
    - 3661 -> "1:01:01"

    ## parseTranscriptMarkdown
    - null input -> null
    - undefined input -> null
    - empty string -> null
    - markdown without "## Transcript" heading -> null (EXTR-05: NO_TRANSCRIPT)
    - "## Transcript\n**0:31** Hello world." -> [{ seconds: 31, text: "Hello world.", section: undefined }]
    - "## Transcript\n**1:23:45** Long video." -> [{ seconds: 5025, text: "Long video.", section: undefined }] (EXTR-03: H:MM:SS)
    - "## Transcript\n**0:12** * Bullet text." -> text is "Bullet text." (bullet stripped)
    - "## Transcript\n### Intro\n**0:00** First.\n**0:10** Second." -> both segments have section: "Intro" (EXTR-04)
    - "## Transcript\n### Intro\n**0:00** A.\n### Main\n**1:00** B." -> first section "Intro", second section "Main"
    - "## Transcript\n### Just a heading" -> [] (heading exists but no segments)

    ## findActiveSegmentIndex
    - empty array, any time -> -1
    - [0s, 30s, 60s], time -1 -> -1 (before first)
    - [0s, 30s, 60s], time 0 -> 0 (exact match first)
    - [0s, 30s, 60s], time 45 -> 1 (between segments)
    - [0s, 30s, 60s], time 999 -> 2 (beyond last)
    - [0s, 30s, 60s], time 60 -> 2 (exact match last)
  </behavior>
  <implementation>
    RED phase: Create transcript-parser.test.ts with all tests above. Run `pnpm test -- --run src/lib/transcript-parser.test.ts` — all tests MUST FAIL (module does not exist yet).

    GREEN phase: Create transcript-parser.ts implementing:

    1. `TranscriptSegment` interface: `{ seconds: number; text: string; section?: string }`
    2. `timestampToSeconds(ts: string): number` — split on ":", handle 2-part (M:SS) and 3-part (H:MM:SS)
    3. `formatTimestamp(seconds: number): string` — inverse of timestampToSeconds; no leading zero on minutes unless hours > 0
    4. `parseTranscriptMarkdown(markdown: string | null | undefined): TranscriptSegment[] | null`:
       - Return null if input is falsy
       - Return null if no "## Transcript" heading found (EXTR-05)
       - Slice from "## Transcript" heading forward
       - Split into lines, iterate:
         - `###` lines set currentSection (strip `### ` prefix)
         - `##` lines (non-###) are skipped
         - Match regex `/^\*\*(\d+:\d{2}(?::\d{2})?)\*\*\s*(?:•\s*)?(.*)$/` for timestamp lines
         - Push `{ seconds: timestampToSeconds(match[1]), text: match[2].trim(), section: currentSection }`
       - Return segments array (may be empty if heading exists but no matches)
    5. `findActiveSegmentIndex(segments: TranscriptSegment[], currentTime: number): number`:
       - Return -1 if empty or currentTime < first segment
       - Binary search: find last segment where seconds <= currentTime

    Run `pnpm test -- --run src/lib/transcript-parser.test.ts` — all tests MUST PASS.

    REFACTOR phase: Review for clarity. No functional changes expected given the narrow scope.
  </implementation>
